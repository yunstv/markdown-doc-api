## 注销个人签署账号

#### 接口说明
* 适用对象 : 接入电子签的商户
* 请求类型 : POST
* apiCode : esign.account.indiv.delete

#### 请求参数
| 参数 | 是否必填 | 类型 | 最大长度 | 参数描述 |
|:----|:-------:|:-----|:-------|:--------|
| user_id | 是 | String | 32 | 账号唯一标识（手机号或身份证号） |


#### 业务参数请求样例
```json
{
    "user_id": "15000000000"
}
```

#### 应答报文

| 参数 | 类型 | 最大长度 | 参数描述 | 示例值 |
|:----|:----|:--------|:--------|:------|
| data | Boolean | 32 | 返回结果 |  |
