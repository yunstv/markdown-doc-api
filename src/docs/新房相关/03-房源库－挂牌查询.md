## 房源库－挂牌查询

#### 接口说明
* 适用对象 : 渠道参与方
* 请求类型 : POST
* apiCode : loupan.service.query

#### 请求参数
| 参数 | 是否必填 | 类型 | 最大长度 | 参数描述 | 示例值 |
|:----|:-------:|:-----|:-------|:--------|:------|
| etc_sale_no | 是 | String | 32 | ETC挂牌编号 |  |
| etc_biz_type | 否 | String | 32 | ETC业务类型，默认查询项目全部数据 | 1001 |

#### 业务参数请求样例
```
{
    "etc_sale_no" : "112233445566",
    "etc_biz_type" : "1000"
}
```
