## 个人二要素信息比对

#### 接口说明

* 接口描述 ：对个人姓名和身份证号的一致性进行检查校验,仅支持中国大陆信息核验
* 适用对象 : 渠道参与方
* 请求类型 : POST
* apiCode : esign.verify.indiv.base

#### 请求参数
| 参数 | 是否必填 | 类型 | 最大长度 | 参数描述 |
|:----|:-------:|:-----|:-------|:--------|
| name | 是 | String | 32 | 姓名 |
| idNumber | 是 | String | 32 | 身份证号（大陆二代身份证） |


#### 业务参数请求样例
```
{
    "name": "张三",
    "idNumber": "320926195511175276"
}
```

#### 应答报文

| 参数 | 类型 | 最大长度 | 参数描述 | 示例值 |
|:----|:----|:--------|:--------|:------|
| result | Boolean | 32 | 比对结果 |  |
| message | String | 32 | 比对结果信息 |  |
