## 证件图片识别

#### 接口说明

* 适用对象 : 渠道参与方
* 请求类型 : POST
* apiCode : ocr.recognition.license

#### 请求参数
| 参数 | 是否必填 | 类型 | 最大长度 | 参数描述 |
|:----|:-------:|:-----|:-------|:--------|
| license_pic_url | 是 | String | 500 | 证件图片url |


#### 业务参数请求样例
```
{
    "license_pic_url":"http://qiushilian.ess.ejucloud.cn/1.jpg"
}
```
