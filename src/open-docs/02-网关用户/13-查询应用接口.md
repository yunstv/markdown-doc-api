## 查询应用接口 

#### 接口说明


* 请求类型 : POST
* apiCode : app.api.list

#### 请求参数
| 参数   | 是否必填 | 类型   | 最大长度 | 参数描述 | 示例值 |
| :----- | :------: | :----- | :------- | :------- | :----- |
| app_id |    是    | String | 16       | 应用id   |        |

#### 业务参数请求样例
```
{
  "app_id": "78IR5R1S4HYE5Z0O"
}
```

#### 应答报文

| 参数        | 是否必填 | 类型   | 最大长度 | 参数描述 | 示例值 |
| :---------- | :------: | :----- | :------- | :------- | :----- |
| module_code |          | String |          | 模块代码 |        |
| module_name |          | String |          | 模块名称 |        |
| api_code    |          | String |          | 接口代码 |        |
| api_name    |          | String |          | 接口名称 |        |
| description |          | String |          | 接口描述 |        |
