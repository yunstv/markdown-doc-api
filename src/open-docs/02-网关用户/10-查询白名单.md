## 查询白名单 

#### 接口说明


* 请求类型 : POST
* apiCode : ipwhite.list

#### 请求参数
| 参数   | 是否必填 | 类型   | 最大长度 | 参数描述 | 示例值 |
| :----- | :------: | :----- | :------- | :------- | :----- |
| app_id |    是    | String | 16       | 应用id   |        |

#### 业务参数请求样例
```
{
    "app_id": "B9GPMA1Q7648MRRQ"
}
```

#### 应答报文

| 参数   | 是否必填 | 类型   | 最大长度 | 参数描述 | 示例值 |
| :----- | :------: | :----- | :------- | :------- | :----- |
| app_id |    是    | String | 16       | 应用id   |        |
| ip     |    是    | String |          | ip       |        |

```
{
  "request_id": "6afcca7f-6c68-4ff2-8f7a-4c94faa7b9f8",
```
