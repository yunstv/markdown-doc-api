## 查询用户权限

#### 接口说明


* 请求类型 : POST
* apiCode : user.permission.list

#### 请求参数
| 参数   | 是否必填 | 类型   | 最大长度 | 参数描述 | 示例值 |
| :----- | :------: | :----- | :------- | :------- | :----- |
| user_no |    是    | String | 16       | 用户编号   |        |

#### 业务参数请求样例
```
{
  "user_no": "78IR5R1S4HYE5Z0O"
}
```

#### 应答报文

| 参数       | 是否必填 | 类型   | 最大长度 | 参数描述   | 示例值 |
| :--------- | :------: | :----- | :------- | :--------- | :----- |
| group_name |    是    | String | 255      | 权限组名称 |        |
| group_no   |    是    | String | 16       | 权限组编号 |        |
