## 添加白名单 

#### 接口说明


* 请求类型 : POST
* apiCode : ipwhite.add

#### 请求参数
| 参数    | 是否必填 | 类型         | 最大长度 | 参数描述 | 示例值 |
| :------ | :------: | :----------- | :------- | :------- | :----- |
| app_id  |    是    | String       | 16       | 应用id   |        |
| ip_list |    是    | List<String> |          | ip列表   |        |

#### 业务参数请求样例
```
{
    "app_id": "B9GPMA1Q7648MRRQ",
    "ip_list": [
        "192.168.1.2",
        "192.168.3.2",
        "192.167.2.1"
    ]
}
```
