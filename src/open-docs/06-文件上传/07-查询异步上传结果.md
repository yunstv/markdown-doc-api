## 通过url获取文件key

#### 接口说明

* 适用对象 : 渠道参与方
* 请求类型 : POST
* 请求地址: /api/file/async/verify

**域名列表**

| 序号 | 环境 | 请求地址                                  |
| ---- | ---- | ----------------------------------------- |
| 1    | 测试 | http://test-gateway.ejuetc.com/inner/base |
| 2    | 预发 | http://pre-gateway.ejuetc.com/inner/base  |
| 3    | 生产 | http://gateway.ejuetc.com/inner/base      |

#### 请求参数

请求参数类型为String**数组**，值为intermediary_url值


#### 业务参数请求样例
```
{
  "urls": ["http://test-gateway.ejuetc.com/inner/base/api/oss/PK3YVYVE6HROXV8L/IMAGE/-526216709/L2pwZ18yMTA5MTYyMWNhN2VmMDdhNTlj!style0.jpg"]
}
```

#### 应答报文

应答报文类型为**数组**

数组类型如下

| 参数 | 类型 | 最大长度 | 参数描述 | 示例值 |
|:----|:----|:--------|:--------|:------|
| intermediary_url | String | 200 | 临时地址 |  |
| target_url | String |  | 正式地址 | |
| file_key | String |  | 文件key | |
| status | String |  | /** 上传完成 */ FINISHED, /** 上传失败 */ FAILED, /** 上传未完成 */ UNFINISHED, /** 业务异常 */ BUSINESS_EXCEPTION | |


#### 应答样例

```json
{
  "requestId": null,
  "responseCode": "000",
  "responseMsg": "成功",
  "data": [
    {
        "intermediary_url": "http://test-tmhouse-oss.ebaas.com/PK3YVYVE6HROXV8L/IMAGE/-161392729/L2pwZ18yMTA5MTgxN2M4MDhiNDNiZWFkYTYyNjIxMjUyZWVjNmQyMDdhM2MzZDYxZDUzNWI%3D.jpg!style0.jpg",
        "target_url":"",
        "file_key":"",
        "status":""
    }
  ]
}
```

### 返回码
| 返回码 | 描述 | 解决方案 |
|:------|:----|:-----|
| 400 | 参数{paramName}不能为空 | 检查参数名称是否正确，参数是否赋值 |
