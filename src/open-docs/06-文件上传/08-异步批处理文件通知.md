## 异步批处理文件通知

#### 接口说明

* 适用对象 : 渠道参与方
* 请求类型 : POST
* 请求地址: 回调请求

#### 请求参数

| 参数            | 类型   | 最大长度 | 参数描述                                         | 示例值 |
| :-------------- | :----- | :------- | :----------------------------------------------- | :----- |
| batch_id        | String | 80       | 批次编号                                         |        |
| api_code        | String | 50       | 进件接口                                         |        |
| **file_list**   | Array  |          |                                                  |        |
| oss_object_name | String |          | 文件KEY                                          |        |
| target_url      | String | 300      | 正式地址                                         |        |
| status          | String |          | /** 上传完成 */ FINISHED, /** 上传失败 */ FAILED |        |
