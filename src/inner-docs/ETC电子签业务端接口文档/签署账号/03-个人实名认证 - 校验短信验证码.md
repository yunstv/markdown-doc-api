## 个人实名认证 - 校验短信验证码

#### 接口说明

* 适用对象 : 电子签业务端
* 请求类型 : POST
* 请求地址 : /esign/api/auth/individual/verifyCode


#### 请求参数
| 参数 | 是否必填 | 类型 | 最大长度 | 参数描述 | 示例值 |
|:----|:-------:|:-----|:-------|:--------|:------|
| etc-token | 是 | String | 32 | etc访问令牌，header |  |
| etc-sys-id | 是 | String | 32 | 系统类型（固定E02），header | E02 |
| esignAccountId | 是 | String | 32 | e签宝账号 |  |
| verifyCode | 是 | String | 32 | 短信验证码 |  |


#### 业务参数请求样例
```
{
	"esignAccountId": "77789",
	"verifyCode": "123456"
}
```


#### 应答报文

| 参数 | 类型 | 最大长度 | 参数描述 | 示例值 |
|:----|:----|:--------|:--------|:------|
| data | Boolean | 32 | 返回结果 |  |


#### 应答样例

```
{
    "requestId": "42dd6c6e-247e-487f-87e8-dd97067b877b",	
    "responseCode": "000",
	"responseMsg": "成功",
	"data": true
}
```
