## 管理中台－流程单详情查询

#### 接口说明
* 适用对象 : 管理中台
* 请求类型 : GET
* 请求路径 : /mgt/trade/mmp/bill/detail/query

#### 请求头
| 参数           | 是否必填 | 类型   | 参数描述    | 示例值                    |
| :------------- | :------: | :----- | :---------- | :------------------------ |
| X-Access-Token |    是    | String | token       |                           |
| etc-sys-id     |    是    | String | ETC系统标识 | 管理中台:E01，电子签：E02 |

#### 请求参数
| 参数 | 是否必填 | 类型 | 最大长度 | 参数描述 | 示例值 |
|:----|:-------:|:-----|:-------|:--------|:------|
| etc_process_bill_no | 是 | String | 32 | ETC流程单编号 |  |
