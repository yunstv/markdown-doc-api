## 房源库－城市区域查询

#### 接口说明

* 适用对象 : 渠道参与方
* 请求类型 : GET
* 请求路径: /mgt/loupan/service/mmp/queryCode

#### 请求参数

无

#### 应答报文

| 参数          | 是否必填 | 类型   | 最大长度 | 参数描述 | 示例值 |
| :------------ | :------: | :----- | :------- | :------- | :----- |
| list          |    是    | List   |          |          |        |
| city_cd       |    是    | String | 20       | 城市编码 |        |
| city_name     |    是    | String | 255      | 城市名称 |        |
| districtList  |    是    | List   |          |          |        |
| district_cd   |    是    | String | 20       | 区域编码 |        |
| district_name |    是    | String | 255      | 区域名称 |        |
