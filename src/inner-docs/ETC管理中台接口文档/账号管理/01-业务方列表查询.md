## 账号管理－业务方列表查询

#### 接口说明

* 适用对象 : 管理中台
* 请求类型 : POST
* 请求路径 : /mgt/partner/api/org/listPartner

#### 请求参数
无


#### 应答报文

| 参数         | 是否必填 | 类型   | 参数描述   | 示例值 |
| :----------- | :------: | :----- | :--------- | :----- |
| partner_name |    是    | String | 业务方名称 |        |
| partner_no   |    是    | String | 业务方编号 |        |
