## 账号管理－公司详情查询

#### 接口说明

* 适用对象 : 管理中台
* 请求类型 : GET
* 请求路径 : /mgt/partner/api/org/getCompanyDetail


#### 请求头
| 参数           | 是否必填 | 类型   | 参数描述    | 示例值                    |
| :------------- | :------: | :----- | :---------- | :------------------------ |
| X-Access-Token |    是    | String | token       |                           |
| etc-sys-id     |    是    | String | ETC系统标识 | 管理中台:E01，电子签：E02 |

#### 请求参数
| 参数     | 是否必填 | 类型   | 参数描述 | 示例值 |
| :------- | :------: | :----- | :------- | :----- |
| companyNo |    是    | String | 公司编号 |        |

#### 业务参数请求样例

/partner/api/org/getCompanyDetail?companyNo=7e7e1f69473441a69dac0934ab2fafd7
