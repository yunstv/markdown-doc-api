#### 接口说明

* 请求类型 : GET
* 请求地址 : /oauth/api/account/activate



#### 请求参数
| 参数  | 是否必填 | 类型   | 最大长度 | 参数描述 | 示例值 |
| :---- | :------: | :----- | :------- | :------- | :----- |
| token |    是    | String |          | token    |        |

#### 请求样例

/oauth/api/account/activate?token=1231231213
