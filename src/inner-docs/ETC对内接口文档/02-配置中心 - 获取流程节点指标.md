## 配置中心 - 获取流程节点指标

#### 接口说明
* 适用对象 : 流程单模块
* 请求类型 : POST
* 请求地址 ：/api/processNode/getIndicators

#### 请求参数
| 参数 | 描述 | 类型 | 是否必填 | 字段说明 |
|:----|:-----|:----|:-----|:----|
| processCode | 流程code         | String | 是 |  |
| processNodeCode | 流程节点code | String | 否 | 为空时查询所有流程节点指标 |

#### 请求样例
```json
{
    "processCode" : "BP100001",
    "processNodeCode" : "BPN100001001"
}
```

#### 应答报文
| 参数 | 描述 | 类型 | 字段说明 |
| :---- | :---- | :----- | :---- |
| processNodeCode | 流程节点code | String |  |
| indicatorList | 流程节点指标集合 | List |  |
| indicatorName | 指标名称 | String |  |
| indicatorCode | 指标code | String |  |
| indicatorRequired | 指标是否必填 | Boolean |  |
| indicatorMaxLength | 指标最大长度 | Integer |  |
| indicatorMinLength | 指标最小长度 | Integer |  |
| indicatorScaleLength | 指标小数精度 | Integer |  |
| indicatorDesc | 指标描述 | String |  |
| indicatorRegex | 指标正则表达式 | String |  |
