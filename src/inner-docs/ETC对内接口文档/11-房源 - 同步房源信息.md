## 房源 - 同步房源信息

#### 接口说明
* 适用对象 : 
* 请求类型 : GET
* 请求地址 ：/service/synRoomInfo

#### 请求参数

无

#### 请求样例
> /service/synRoomInfo

#### 应答报文

返回Boolean值

#### 应答样例
```json
{
    "requestId": null,
    "responseCode": "000",
    "responseMsg": "成功",

```
