## 运营平台－根据用户手机号查询邮箱+工号

#### 接口说明
* 适用对象 : 管理中台
* 请求类型 : POST
* 接口code : omp.user.phone2mail

#### 请求参数
| 参数 | 是否必填 | 类型 | 最大长度 | 参数描述 | 示例值 |
|:----|:-------:|:----|:--------|:--------|:------|
| mobileList | 是 | Array |  | 员工手机号 |  |


#### 响应参数
| 参数 | 是否必填 | 类型 | 最大长度 | 参数描述 | 示例值 |
|:----|:-------:|:----|:--------|:--------|:------|
| phoneMail | 是 | Map |  | 手机邮箱映射 |  |
| key | 是 | String |  | 手机 |  |
| value | 是 | Object |  |  |  |
| emial | 是 | String |  | 邮箱 |  |
| workNo | 是 | String |  | 工号 |  |


#### 请求样例
```json
{
  "mobileList": ["17717917547"]
}
```


#### 应答样例
```json
{
  "success": true,
  "code": 0,
  "message": "成功",
  "timestamp": 1651738060088,
  "result": {
    "phoneMail": {
      "17717917547": {"email":"zhaotao@ehousechina.com", "workNo":"130913"}
    }
  }
}
```
